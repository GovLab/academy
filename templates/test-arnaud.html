Now we want to iterate over the people we have.
    {% for person in people %}
  • {{person.name}}
    • {% endfor %}

What about defining a mini-template to render each person. We can use a macro for that.

{% macro showPerson(person) -%}
{{person.name}}
{%- endmacro %}
    {% for person in people %}
  • {{showPerson(person)}}
    • {% endfor %}

Now what about making it easy to just grab one user.

{{ people|selectattr("name", "equalto", "Beth Simone Noveck")|first }} {{ showPerson(people|selectattr("name", "equalto", "Beth Simone Noveck")|first) }}
    {% for p in people|selectattr("name", "equalto", "Beth Simone Noveck") %}
  • {{p}}
    • {% endfor %}
{% macro showPersonByName(name) -%} {{ showPerson(people|selectattr("name", "equalto", name)|first) }} {%- endmacro %} {{ showPersonByName('Beth Simone Noveck')}} {{ showPersonByName('Arnaud Sahuguet')}}
